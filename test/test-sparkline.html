Sparklines - Test of on-demand jquery and plugin loading

No jQuery scripts or plugins are present in this page. All necessary javascripts are loaded on-demand. The following sparklines are dynamically created based on values retrieved from the SDMX REST API.

This phrase shows the evolution of unemployment in Germany , France and Spain from 1995-2005.

    var de = { locations: "DEU", subjects: "UNEMP", frequency: "A", params: { startTime: 1995, endTime: 2005 }};

    var fr = { locations: "FRA", subjects: "UNEMP", frequency: "A", params: { startTime: 1995, endTime: 2005 }};
    SDMX.insertSparkline( fr, "fr_unemp" );
    var es = { locations: "ESP", subjects: "UNEMP", frequency: "A", params: { startTime: 1995, endTime: 2005 }};
    SDMX.insertSparkline( es, "es_unemp" );
});
