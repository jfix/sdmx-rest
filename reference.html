On this page you'll find the values that you need to create useful requests to the OECD's SDMX REST API. All (and only) the variables, locations and time periods are listed. However, I suppose that this is only a very small subset of the actually available methods. In consequence, this page should change and enrich itself over time.

Open the section below by clicking on its heading. Please note that contents is dynamically requested from the API, so it may take some time, in particular for voluminous data.

For information, the information is retrieved using the GetDataStructure method.

  • Concepts (this defines the following metadata)
  • Subjects (the indicators, if you like another word for the same thing)
  • Locations (country, area and region codes as they may not correspond to ISO codes)
  • Statuses (statistical information for a value)
  • Time periods (how to express the time periods you want)
  • Frequencies (availability of data)
